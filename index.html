Librerías de R

econocharts - Permite crear gráficos microeconómicos como curvas de oferta y demanda, curvas de indiferencia, fronteras de posibilidades de producción, entre otros, así como calcular los puntos de equilibrio.
calendR - Basado en ggplot2, permite crear calendarios anules y mensuales para la visualización de datos mediante mapas de calor o para la organización de tareas.
DTDA.cif - Estimador no paramétrico de incidencias acumuladas de riesgos competitivos bajo doble truncamiento.
DiscreteQvalue - q-valor para pruebas discretas uniformes y homogéneas.
TwoSampleTest.HD - Contraste de dos muestras para la igualdad de distribuciones para datos de alta dimensión.
Equalden.HD - Contraste de igualdad de un conjunto de densidades de alta dimensión.
DTDA.ni - Estimador no iterativo para la distribución acumulada de una variable doblemente truncada.
